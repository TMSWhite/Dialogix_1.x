Required Software for the Boricua Youth Study (BYS) Intallation Instructions

 

THE FOLLWING SOFTWARE IS PRE-REQUIRED ON ANY COMPUTERS USED FOR THE ADMINISTRATION OF THE BYS COMPUTERIZED INSTRUMENTS UTILIZING A LIMITED RELEASE VERSION OF THE Dialogix SYSTEM SOFTWARE:

(1) Install the Java Runtime Environment (JRE) 1.3.0 on the machine:

  • Check which (if any) version of version is already installed
  • open a DOS window. You will see the command prompt:

    C:\>

  • type java -version followed by the Enter key.
    • If you do not have a JRE installed, you should see a message like this:

      C:\>java -version
      Bad command or file name

    • If you do have Java already installed, you should see a message which tells you which version is intalled:

      C:\>java -version
      java version "1.3.0"
      Java(TM) 2 Runtime Environment, Standard Edition (build 1.3.0-C)
      Java HotSpot(TM) Client VM (build 1.3.0-C, mixed mode)

  • If you do have JRE installed, and it is not version 1.3.0, you must remove it before installing the new version, otherwise version conflicts can occur that will prevent the study from running properly.
    • Uninstall previous versions of JRE from from the Add/Remove Programs option within the Control Panel folder (press Start, then Settings, then Control Panel, then double click on Add/Remove Programs).
    • Previous versions will be listed as something like Java 2 Runtime Environment Standard Edition v1.2.
    • The newest version is listed as Java 2 Runtime Environment Standard Edition v1.3 (or International Edition v1.3)
  • If you do not have have the current JRE installed, run j2re1_3_0-win-i.exe and follow the instructions, selecting the default choices.

(2) Is Netscape installed on your machine? If not, do the following:

  • Run cc32d473.exe and follow the instructions, selecting the default choices.

(3) Do you have Acrobat Reader installed on your machine? If not, install the appropriate reader (e.g. English 32 bit (Win95/98) or 16 bit, Spanish 32 bit (Win95/98) or 16 bit) from the Acrobat Folder on the CD.


FOR DATA MANAGEMENT COMPUTERS ONLY:

Repeat Steps 1 and 2 and 3 from above and continue.

(4) Do you have Active Perl installed on your machine? If not, run APi522e.exe from the CD and follow the instructions.
