Configuring Intranet Servers for BPRS & GAF

Introduction:

  • Patient data can not be transmitted over the web, due to existing HCFA standards and proposed HIPPA regulations.
  • However, it is legal to use the BPRS and GAF applications on a local machine, or a secure intranet, if and only if your local facility has certified that no unauthorized users can access the patient information.
  • Below are instructions for preparing a single machine or local intranet for using my BPRS and GAF applications.

Methods:

  • See the powerpoint presentation for a discussion of the reasons and methods for optimizing the BPRS and GAF for clinical use.
  • When data is submitted from the forms, a Java Servlet processes the information, calculates the overall and subscale scores, and saves all of the information to a comma-delimited text file in the Sevlet's home directory. Thus, there is no need for a proprietary database - the comma-delimited text files can be imported directly into Excel, Access, or other tools for analysis.

Requirements:

  • Secure Intranet
    • Patient data can not be sent over the web electronically (without first consulting HCFA).
    • The data files containing the stored BPRS and GAF data must not be accessible to unauthorized users.
    • N.B. The proposed HIPPA regulations impose significant penalties for violating these regulations. Although these guidelines are designed to regulate industries which might want to profit from knowledge of a patient's health status, they do also regulate clinical and research activity. HIPPA has a 60 day public commentary period on these regulations. Since they will directly impact upon pyschiatric and forensic care, it is worth reviewing and commenting upon these proposals.
  • Web Server
    • Windows NT + IIS or Personal Web Server
    • Linux + Apache
    • Unix + Apache
  • Java Servlet Support
    • Windows - Sun's JSWDK (JavaServer Web Development Kit)
    • Unix - See Apache pages for native JavaServlet support. Otherwise use JSWDK
      • for JSWDK, if you follow the setup directions, you will be able to point your browser to working example pages.
  • Java Support
    • Windows/Unix - Sun's JDK1.2
  • BPRS/GAF source files
    • Please contact me.
    • Once you have the source files, follow the instructions in the Readme file to install, test, and troubleshoot BPRS and GAF.
