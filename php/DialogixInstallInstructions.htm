Dialogix Install Instructions for OMH

Required Software

  • Java -- needed to run Dialogix, since Dialogix is written in Java
  • Dialogix Installer File for OMH -- the Dialogix program and support files
  • Excel -- for editting instruments

Optional Software

  • Perl 5.6.1.638 -- needed for add-on debugging and data management utilities
  • TortoiseCVS -- a version control system for tracking changes
  • TkDiff -- a graphical viewer of differences between versions of instruments

Documentation

  • Technical Instruction Manual (out of date, but does provide all details of Dialogix syntax)
  • Microsoft Help File (incomplete)
  • Authoring Instruments from user's perspective (draft)
  • Overview of security for IRB (for Wave6 Study @ NYPI)

Presentations

  • HIV Center Grand Rounds (March 28, 2002)
  • 2002 Institute for Mental Health Management (December 3, 2002)
  • Overview for DOH (November 3, 2004)

Running Dialogix

  • run C:\usr\local\dialogix\dialogix.bat (there may be a Dialogix shortcut on the desktop)

Editing Dialogix Files

  • Instruments are located in C:\usr\local\dialogix\webapps\OMH\WEB-INF\schedules\*.txt
